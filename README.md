# employee-management-system-dll-c
A menu-driven Employee Record Management System implemented in C using Doubly Linked List (DLL). This project supports insertion, deletion, forward and reverse traversal, and demonstrates Deque (Double Ended Queue) operations.

This project implements an **Employee Record Management System** using **Doubly Linked List (DLL) in C**. It supports efficient insertion, deletion, traversal in both directions, and demonstrates **Deque operations**.
